# Eye Blink Detection

A Python and OpenCV-based application designed to detect eye blinks in real-time using a webcam feed.

## Features
* Real-time eye blink tracking.
* Smooth UI built with Flask (templates and static assets).
* Eye Aspect Ratio (EAR) calculation.
